Behind Story##

The most challenge part was 'Passing search result departure and destination both values together' when Storing a search result value into Redux store as an object like below.

const searchValues = {{ id: 1, departure: YUL, destination: ICN }, {...} };

For instance, to get 'destination value of Trip1' when saving 'departure value of Trip2', I first needed to check if 'the object of Id 1' exists in Redux store. If it doesn't, define the destination value empty.

I solved it by using normalized data structure, creating an array of 'Trip Ids' like below. So that I can quickly check if 'Trip1' object exists with findIndex() method.

const tripIds = [ 1, 2, ... ];
